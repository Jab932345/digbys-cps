Setup guide
  1. Connect Brother QL-820NWBc to kitchen Wi-Fi via its LCD menu
  2. Hold the feed button to print a network report — note the IP address
  3. Enter that IP above and save
  4. Set up Raspberry Pi print server — ask Claude to generate the code
  5. Every device on the same Wi-Fi sends labels to the one printer

How to restore from backup

If data is lost or incorrectly changed, go to supabase.com → your project → Table Editor → select the table → Import data from CSV. Upload your backup file and the data will be restored. The print history audit log cannot be altered by any staff member.
